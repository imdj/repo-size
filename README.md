# Repo size

A simple website that returns the size of a repo before you clone it. It supports GitHub, Bitbucket, and Gitea. Access it at [repo-size.imadij.com](https://repo-size.imadij.com)

Note: Gitlab is not supported because it requires authentication to access repository size statistics.

## Developing

The website is built with SvelteKit and Tailwind CSS.she

1. Install dependencies

```sh
npm install
```

2. Run the development server

```sh
npm run dev
```

3. Build the website for production

```sh
npm run build
```
